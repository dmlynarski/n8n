# Przewodnik po n8n Workflows

Repozytorium zawierające kompleksowy przewodnik po pracy z workflowami n8n.

## Szybki start

1. Zainstaluj n8n (Docker, npm lub Cloud)
2. Utwórz nowy workflow
3. Dodaj trigger node
4. Zbuduj logikę z kolejnych node'ów
5. Testuj i aktywuj

Szczegóły znajdziesz w przewodniku.
